## Подсказка по GIT

### Создание репозитория
``` sh
git init
```
### Добавить изменения
``` sh
git add
```
### Сохранение
``` sh
git commit -m "Message"
```
### Журнал изменений
``` sh
git log
```
### Журнал изменение (кратко)
``` sh
git log --oneline
```
### Переход к изменениям
``` sh
git checkout
```
